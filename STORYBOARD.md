# Agent IC v16 submission storyboard

Target length: **≤ 90 s**, final render **84 s**. Recorded at 1920×1080/30 fps.

Audio source: `demo-out/agent-ic-audio-mastered-v13.wav` (Kokoro TTS, mastered to -14 LUFS, 48 kHz stereo), trimmed/faded to match the final 84 s cut.  
Base recording: `demo-out/agent-ic-demo-v16-raw.webm` — a single-take, live browser screencast captured by Playwright at 1920×1080. The browser shows the real Agent IC app plus real proof terminal pages served from `demo-out/terminals-v16/`.  
Post-production: `scripts/post-produce-v16.mjs` transcodes to h264, burns intro/outro cards and captions, and trims/fades audio.  
Final artifact: `demo-out/agent-ic-demo-v16.mp4` (promoted to `demo-out/agent-ic-demo-final.mp4`).

Narrative arc: **Hook → Problem → Proposal/Onboard → Evaluate → Fund → Govern (403 block + red vignette) → Decide (artifact shot) → CTA**.
